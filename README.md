# Funnel Analytics Mini App

A mini app for visualizing and analyzing campaign funnels, built with Vue 3, TypeScript, and Tailwind CSS.

## How to run

Make sure you have at least **Node.js ^20.19.0 or >=22.12.0**.

After cloning the repo, run the following commands in the root directory:

```sh
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
